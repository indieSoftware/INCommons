INCommons

This library provides common functionalities every ios app can use.

Installation

To include via CocoaPods add to the Podfile:

pod 'INCommons'

Carthage

To include via Carthage add to the Cartfile:

github "indieSoftware/INCommons"

SPM

To include via SwiftPackageManager add the repository:

https://github.com/indieSoftware/INCommons.git
