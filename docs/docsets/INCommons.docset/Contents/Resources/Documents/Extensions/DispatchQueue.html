  • once(token:block:)

    Executes a block of code, associated with a unique token, only once.

    The code is thread safe and will only execute the block once even in the presence of multithreaded calls.

    Source: https://stackoverflow.com/questions/37886994/dispatch-once-after-the-swift-3-gcd-api-changes

    Declaration

    Swift

    class func once(token: String, block: () -> Void)

    Parameters

    token

    A unique token as key for the block. Use a unique reverse DNS style name such as com.vectorform. or a GUID.
    block

    Block to execute once.
